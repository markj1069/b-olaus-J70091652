1.2 Links

There are various types of links you can use. A simple link to a URL with https://github.com and optionally with it a title: GitHub. An email address me@me.me. A node in this document, See Introduction. Always use the exact node name for that one. xref will include the text “see” before the link. To insert something different, use see Introduction (“See”) or See Introduction (nothing is inserted). With an additional argument, you can change the text of the link, See this introduction.

It is possible to link to external manuals with these commands by adding more arguments, as in , See Overview in Texinfo’s manual for the complete reference on Texinfo!
